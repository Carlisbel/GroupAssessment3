Assessment 3 - Team Project Resume Builder - PROresume4U.com.au

Project PLan

- Project Page from assessment 2 - https://carlisbel.github.io/GroupAssessment2/project.html

Firstly we would need to hire the appropriately skilled personal to handle "Advertising and Marketing", "Java and JavaScript programming", an individual who could Manage the "database, templates and accounts" and a "Web development professional" to get it all to work together perfectly. Obtain an available domain name to run the website under.

Stage 1
Landing page, JavaScript login prompt to create a user account, this creates a database entry for the user to save their data too, for the purpose of completing a detailed resume which at a later time can be edited or downloaded. Also, at this stage subscription can be paid at first page entry or at the end once completed their resume they can choose to pay now or later at the end of their resume creation.

Stage 2
A New user will be able to enter their information into a html form each input field will be a section of a resume such as Professional Summary, Education, Work history and Skills etc. This information will get saved into the database in their account name.

Stage 3
The user will get to choose from a range of resume templates found in the database and preview their resume with their data inserted before saving it in their chosen format .doc or .pdf on their personal device. “Prior to being able to save and download this document they must have satisfied the subscription terms to continue to the download page”
